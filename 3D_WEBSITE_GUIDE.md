## Step 1 — Install Claude Code

Claude Code is the CLI that turns Claude into your terminal-based engineer. It reads your codebase, writes files, runs commands, and ships production code.

**Official guide:** https://docs.claude.com/en/docs/claude-code/setup

**Run this in your terminal:**

```bash
npm install -g @anthropic-ai/claude-code
```

**Then start it inside any project folder:**

```bash
claude
```

Sign in when prompted. Done.

---

## Step 2 — Install Framer Motion

Framer Motion is the animation library Claude Code uses to make every interaction smooth, layered, and high-end. Without it, your sites look static. With it, they look like the $10K agency build.

**Docs:** https://motion.dev

**Inside your project folder, run:**

```bash
npm install framer-motion
```

That's it. Claude Code will detect it and start using it automatically when you ask for animations.
